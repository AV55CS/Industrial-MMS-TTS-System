# Usage Guide

## Basic Usage

1. **Enter Text**: Type your message in Hindi or English
2. **Select Type**: Choose message type or use "Auto"
3. **Select Language**: Choose language or use "Auto-detect"
4. **Generate**: Click "Generate Simple TTS"

## Features

### Number Conversion
- `1234` → "one thousand two hundred thirty four"
- `₹50,000` → "fifty thousand rupees" 
- `12:30` → "twelve thirty"

### Voice Effects
- **Safety**: Louder, repeated
- **Urgent**: Loudest, fast-paced
- **Instructions**: Clear, methodical

### Sample Texts
Use the dropdown to load pre-made examples for testing.
